Primer Componente

Un componente es una pieza de intefaz grafica (UI) que contiene su propia funcionalidad, de tal forma es un pedazo dentro de una aplicacion. En otra palabras son funciones de Javascrips que devuelve markup (marcado) a la pagina de donde se importe. para utilizar React hay que tener instalado los Snippets de visual estudio, por ejemplo para crear un funcion componente se escribe ffc 
        
          function PrimerComponente() {
            return ( <h1>Hola mundo</h1> );
          }
        
          export default PrimerComponente;
para llamar a nuestro PrimerComponente tenemos que importarlo desde nuestro archivo main.jsx, quedando de la siguiente forma: 
      
        import React from 'react'
        import ReactDOM from 'react-dom/client'
        import ComponentePrueba from './ComponentePrueba'

        ReactDOM.createRoot(document.getElementById('root')).render(
          <React.StrictMode>
            <ComponentePrueba />
          </React.StrictMode>,
        )
con el atajo de teclado alt + Shift + F (Formatea el docuemento) en visual studio Code.
La forma recomendada en la actualidad es usar el Snippets rafc que crea una funcion flecha 
      
        import React from 'react'

        export const Componente2 = () => {
          return (
            <div>Componente2</div>
          )
        }
Pero cambia la forma de llamardo ahora en el momento de importarlo hay que colocar entre llaves { } quedando el codigo de la siguiente forma: 
      
        import { Componente2 } from './Componente2'

Variables

Nota: Los nombres de los componente debe esta con mayuscula inicial o sale error al momento de importarlo, ademas es una convencion de buenas practicas. para pasar los variables desde un componente al main.jsx para que lo renderise React, hay que escribir la variables entre llaves { }, quedando de la siguiente forma: 
      
        const wordSerntence='frase de prueba'
        const numero=145678
        const tecnologias=['React','Spring boot','node']
        const verdadero=true
        const funcionFlecha=()=>1+1
        const objeto={nombre:'Curso de javascrip',duracion:4}
        const fecha=new Date();
  
  
        export const Variables=()=>{
          return (
            <h1>{wordSerntence}</h1>
          )
        }
los campos primitivos como string, numeros, boleanos y array se pasan sin ningun problemas, las funciones deben colocar los parentesis para que se ejecute, ejemplo: 
      
        const funcionFlecha=()=>1+1
 
  
        export const Variables=()=>{
          return (
            <h1>{funcionFlecha()}</h1>
          )
        }
En cuanto a los objetos , los objeto tipo fecha y los JSON sale error para poder enviarlos hay que converirlos primeros a string usando stringify ejemplo: 
      
        const objeto={nombre:'Curso de javascrip',duracion:4}
        const fecha=new Date();

        export const Variables=()=>{
          return (
            <h1>{JSON.stringify(objeto)}</h1>
          )
        }

Fragmentos

Si notamos en los ejemplos anteriores en caso que queremos devolver varias variables tendriamos un error, porque para poder enviarlo hay que enviarlo envuelto desde un padre, envolviendo dentro de un Div, quedando el codigo de la siguiente forma: 
      
        const wordSerntence='frase de prueba'
        const numero=145678
        const tecnologias=['React','Spring boot','node']
        const verdadero=true
        const funcionFlecha=()=>1+1
        const objeto={nombre:'Curso de javascrip',duracion:4}
        const fecha=new Date();

        export const Fragmento=()=>{
          return (
              <div>
                <h1>Las variables de JSX son:</h1>
                <h4>Variables tipo String</h4><p>{wordSerntence}</p>
                <h4>Variables tipo numerico</h4><p>{numero}</p>
                <h4>Variables tipo Array</h4><p>{tecnologias}</p>
                <h4>Variables tipo boleano</h4><p>{verdadero}</p>
                <h4>Variables tipo funcion</h4><p>{JSON.stringify(funcionFlecha())}</p>
                <h4>Variables tipo objeto</h4><p>{JSON.stringify(objeto)}</p>
              </div>
          )
        }
Esa era la forma antigua en que se hacia por eso es que aparecieron los Fragment quedanto de la siguiente forma: 
      
        const wordSerntence='frase de prueba'
        const numero=145678
        const tecnologias=['React','Spring boot','node']
        const verdadero=true
        const funcionFlecha=()=>1+1
        const objeto={nombre:'Curso de javascrip',duracion:4}
        const fecha=new Date();

        export const Fragmento=()=>{
          return (
              <Fragment>
                <h1>Las variables de JSX son:</h1>
                <h4>Variables tipo String</h4><p>{wordSerntence}</p>
                <h4>Variables tipo numerico</h4><p>{numero}</p>
                <h4>Variables tipo Array</h4><p>{tecnologias}</p>
                <h4>Variables tipo boleano</h4><p>{verdadero}</p>
                <h4>Variables tipo funcion</h4><p>{JSON.stringify(funcionFlecha())}</p>
                <h4>Variables tipo objeto</h4><p>{JSON.stringify(objeto)}</p>
              </Fragment>
          )
        }
Tambien se puede cambiar la etiqueta Fragment por simplemento < >

props propiedades

se utlizan para pasar informacion de un componente padre a un componente hijo.
vamos a crear un componente llamado PrimerComponente.jsx, tenemos que utilizar el snippets rafc
y nos genera el siguiente codigo: 
      
        import React from 'react'

        export const PrimerComponente = () => {
          return (
            <div>PrimerComponente</div>
          )
        }
dentro de los parentices se pasa los props y desde el main.jsx se eviand el objeto con los atributos titulos y subtitulos de la siguiente forma: 
      import React from 'react'
      import ReactDOM from 'react-dom/client'
      import './style.css' 
      import { PrimerComponente } from './PrimerComponente'

      ReactDOM.createRoot(document.getElementById('root')).render(
        <React.StrictMode>
          <PrimerComponente titulo="Este es un titulo" subtitulo="subtitulo de prueba"/>
          </React.StrictMode>,
      )
al componente le quitmos al reurno los Div y dejamos simplemente los < > quedando de la siguiente forma: 
      
        import React from 'react'

        export const PrimerComponente = (props) => {
        return (
          <>
            <h1>props:</h1>
          </>
          )
        }
ahora lo siguiente hay que desestructurar quitando la palabra prop y colocamos cada uno de los atributos entre llaves { } 
      
        import React from 'react'

        export const PrimerComponente = ({titulo, subtitulo}) => {
        return (
          <>
            <h1>El titulo es:{titulo} </h1>
            <h1>El subtitulo es :{subtitulo} </h1>
          </>
          )
        }
en caso que el segundo parametro sea un numero, por decir algo un contador y dentro del componente hay que incrementarle uno, entonces hay que especificar que sea numerico y en caso de pasarle un caracter hay que restringirlo, de la siguiente forma:
en el main.jsx hay que quitar las comillas al pasar el subtitulos y colocarlos entre llaves { } 
      import React from 'react'
      import ReactDOM from 'react-dom/client'
      import './style.css' 
      import { PrimerComponente } from './PrimerComponente'

      ReactDOM.createRoot(document.getElementById('root')).render(
        <React.StrictMode>
          <PrimerComponente titulo="Este es un titulo" subtitulo={10}/>
          </React.StrictMode>,
      )
y el componente que lo recibe quedaria el codigo así : 
      
        import React from 'react'

        export const PrimerComponente = ({titulo, subtitulo}) => {
        return (
          <>
            <h1>El titulo es:{titulo} </h1>
            <h1>El subtitulo es :{subtitulo+1} </h1>
          </>
          )
        }
